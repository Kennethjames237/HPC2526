# High Performance Computing Labs

## What is High Performance Computing?

High Performance Computing (HPC) refers to the use of powerful computing systems and parallel processing techniques to solve complex computational problems efficiently. It is widely used in scientific research, engineering, data analysis, simulations, and artificial intelligence.

## About This Repository

This repository collects the laboratory exercises completed during the High Performance Computing course. It includes practical activities designed to explore core concepts of parallel computing, system performance, and computational efficiency.

The purpose of this repository is to document the learning process and keep all lab work organized in one place.
